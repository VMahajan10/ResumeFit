## Key Files Explained

### Configuration
- **package.json**: npm dependencies and build scripts
- **tsconfig.json**: TypeScript compiler options
- **vite.config.ts**: Vite bundler configuration for building the extension
- **manifest.json**: Chrome extension manifest (Manifest V3)

### Source Code
- **src/types.ts**: Shared TypeScript interfaces and types
- **src/background.ts**: Background service worker - handles extension lifecycle, message passing, state storage
- **src/contentScript.ts**: Content script injected into pages - creates sidebar iframe, manages visibility
- **src/sidebar/sidebar.html**: HTML structure for the sidebar UI
- **src/sidebar/sidebar.css**: Styles for the sidebar
- **src/sidebar/sidebar.ts**: Sidebar logic - UI interactions, state management, message handling

### Build Output
All files in `dist/` are generated by running `npm run build`. Load the `dist/` folder as an unpacked extension in Chrome.
